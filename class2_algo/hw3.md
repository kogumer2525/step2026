# 宿題3

## 問題

次の操作をほぼ O(1) で実現するデータ構造を考える
- 与えられた <URL, Web ページ> があるかないかを検索
- もしない場合、キャッシュ内で一番古い <URL, Web ページ> を捨てて、代わりに与えられた <URL, Web ページ> を追加する

## 自分の考え

### データ構造

- ハッシュテーブル：
  - key(url), value(順序を管理するための連結リストへのポインタ), next（ハッシュ衝突時用）
- 順序を管理するための連結リスト：
  - url, contents（Webページ）, prev（一つ前のリスト）, next（一つ後ろのリスト）
  - O(1)で挿入・削除をするためにリスト先頭にhead, 末尾にtailを入れておく
